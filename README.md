# Лабораторная работа №1. Источники задач и контракты

## Запуск uv
```
uv sync
uv venv
```
Для Windows
```
.venv\scripts\Activate
```
Для Linux/iOS
```
source .venv\Scripts\activate
```

# Источники задач
Реализованы как классы в `src\source.py`
- API
- Генератор
- Чтение из файла

# Задачи и контракт
`src\task.py`
Task представляется через dataclass. Для любого источника задача имеет:
- id — уникальный идентификатор задачи.
- payload — произвольные данные задачи, зависят от источника.

У всех задач есть метод `get_tasks()`, проходящий через контракт с помощью **Protocol** и **runtime_checkable**

# Запуск тестов
через pytest
```
pytest .
```
Функция `main()` в файле `src\main.py` позволяет с помощью генератора создавать некоторое количество задач на вывод для примера.
